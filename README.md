# Логистическое веб-приложение

Бета-версия веб-приложения для логистики (локальная, без сервера).

## Возможности
- Создание заказов (выбор позиций и количества)
- Распределение груза по поставщикам
- Финансовая часть: предоплата / погашение
- Справочники: позиции и поставщики
- Аудит действий

## Технологии
- React + TypeScript + Vite
- Tailwind CSS
- LocalStorage для хранения данных
